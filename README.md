# Numeric-multiples or not
### Is that number multiples or not?

```python
x=1 #x는 1이다 

while x<=15: #x가 15가 될때까지 while 구문을 반복

if x%2==0: #%연산자는 나머지를 계산, x를 2로 나누었을 때 나머지가 0이면
--- x를 2로 나누었을 때 나머지가 0인 즉, 짝수인 수들만 해당---
if x%6==0: #2로 나누었을 때 나머지가 0이면서 
6으로 나누었을 때의 나머지도 0인 경우

print(x,"는 6의 배수입니다.") #숫자 는 6의 배수입니다 -> 출력

elif x%3==0: #2로 나누었을 때 나머지가 0이지만
 6으로 나누었을 때의 나머지는 0이 아님,
그러나 3으로 나누었을 때 나머지는 0이다

print(x,"는 3의 배수입니다.")

else: #2로 나누었을 때 나머지=0,
 6으로 나누면 나머지<>0,
 3으로 나누면 나머지<>0

print(x,"는 아무것도 아닙니다.")

x=x+1 #while 구문에 사용되는 x가 1씩 커진다 (1~15)

```
